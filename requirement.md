# Project Title
AI-Powered Nearby Pharmacy Finder & Medicine Ordering System

## 1. Problem Statement
Accessing nearby pharmacies during emergencies or late hours is difficult due to lack of real-time information about operational status, medicine availability, and ordering options. Users often waste time searching for open pharmacies or calling multiple stores.

## 2. Objective
To build a GPS-enabled intelligent platform that helps users:
- Discover nearby pharmacies
- View real-time open/closed status
- Check timings and contact details
- Place and track medicine orders
- Receive AI-powered recommendations

## 3. Functional Requirements

### 3.1 User Module
- User registration & login
- Enable GPS location detection
- Search nearby pharmacies
- View pharmacy details (timings, contact, status)
- Place medicine order
- Track order status
- Receive notifications

### 3.2 Pharmacy Module
- Pharmacy registration & login
- Update operational status (open/closed)
- Manage medicine availability
- Accept/reject orders
- Update order status

### 3.3 AI Features
- Smart pharmacy ranking
- Demand prediction
- Alternative medicine suggestions
- Emergency prioritization

## 4. Non-Functional Requirements
- Real-time performance
- Secure authentication (JWT)
- Scalable backend
- Mobile-friendly UI
- Data privacy & encryption

## 5. Technologies Used
- Frontend: React.js / React Native
- Backend: Node.js + Express
- Database: MongoDB / MySQL
- Maps: Google Maps API
- Notifications: SMS API
- AI: Python ML models / AI APIs
